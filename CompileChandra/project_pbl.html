Keywords in C++

                    
Keywords are reserved words that have special meanings in the language.

                    

                        

                            Keyword
                            Description
                            Example
                        

                        

                            int
                            Integer data type
                            int x = 5;
                        

                        

                            if
                            Conditional statement
                            if (x > 0) { ... }
                        

                        

                            for
                            Loop construct
                            for (int i = 0; i < 10; i++) { ... }
                        

                        

                            while
                            Loop construct
                            while (x > 0) { ... }
                        

                        

                            class
                            Class definition
                            class MyClass { ... };
                        

                        

                            return
                            Return statement
                            return value;
                        

                        

                            void
                            No return type
                            void function() { ... }
                        

                        

                            const
                            Constant value
                            const int MAX = 100;
                        

                        

                            static
                            Static storage
                            static int count = 0;
                        

                        

                            namespace
                            Namespace declaration
                            namespace myspace { ... }
                        

                    

                

                
                

                    
Operators in C++

                    
Operators are symbols that tell the compiler to perform specific operations.

                    

                        

                            Operator
                            Type
                            Example
                        

                        

                            +, -, *, /
                            Arithmetic
                            a + b, a - b, a * b, a / b
                        

                        

                            ==, !=, >, <, >=, <=
                            Comparison
                            a == b, a != b, a > b
                        

                        

                            &&, ||, !
                            Logical
                            a && b, a || b, !a
                        

                        

                            =, +=, -=, *=, /=
                            Assignment
                            a = b, a += b, a -= b
                        

                        

                            &, |, ^, ~, <<, >>
                            Bitwise
                            a & b, a | b, a ^ b, ~a
                        

                    

                

                
                

                    
Separators in C++

                    
Separators are symbols used to separate different parts of the code.

                    

                        

                            Separator
                            Description
                            Example
                        

                        

                            { }
                            Braces - Block of code
                            if (condition) { statements; }
                        

                        

                            ( )
                            Parentheses - Function calls, conditions
                            function(param1, param2)
                        

                        

                            [ ]
                            Brackets - Arrays
                            int array[10];
                        

                        

                            ;
                            Semicolon - End of statement
                            int x = 10;
                        

                        

                            ,
                            Comma - Separator for parameters, declarations
                            int x, y, z;
                        

                        

                            .
                            Dot - Member access
                            object.member
                        

                        

                            ::
                            Scope resolution
                            std::cout
                        

                    

                

                
                

                    
Identifiers in C++

                    
Identifiers are names given to entities such as variables, functions, classes, etc.

                    

                        

                            Rule
                            Description
                            Example
                        

                        

                            First character
                            Must begin with a letter or underscore
                            count, _value
                        

                        

                            Subsequent characters
                            Can contain letters, digits, underscores
                            user1, total_sum
                        

                        

                            Case sensitivity
                            Identifiers are case-sensitive
                            Count â‰  count
                        

                        

                            Reserved words
                            Cannot use keywords as identifiers
                            int, class, return (invalid as identifiers)
                        

                    

                

                
                

                    
Literals in C++

                    
Literals are fixed values that are directly written in the code.

                    

                        

                            Type
                            Description
                            Example
                        

                        

                            Integer
                            Whole numbers
                            42, 0, -10, 0xFF (hex), 077 (octal)
                        

                        

                            Floating Point
                            Decimal numbers
                            3.14, 2.5e2, .5, 1.0f
                        

                        

                            Character
                            Single character
                            'a', '5', '\n', '\t'
                        

                        

                            String
                            Sequence of characters
                            "Hello", "123", "", "x"
                        

                        

                            Boolean
                            Logical values
                            true, false
